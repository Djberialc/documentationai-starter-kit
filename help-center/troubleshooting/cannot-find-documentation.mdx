---
title: Cannot Find Documentation Page
description: Fix missing documentation pages quickly.
---

## Symptoms

- Selected page returns 404 error
- Cannot locate a specific documentation section

## Possible Causes

- The page hasn’t been created yet
- Wrong URL was entered
- The page was deleted

<Steps>
  <Step title="Search">
    Use the search bar at the top to find your page.
  </Step>
  <Step title="Check Sidebar">
    Expand all sidebar menus to see if it’s nested.
  </Step>
  <Step title="Create New Page">
    If still missing, create a new page or contact your workspace admin.
  </Step>
</Steps>
